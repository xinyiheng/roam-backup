    - Python是一门面向对象编程语言，它提供了一些支持面向对象编程的语言特性，这些特性有如下明确的特征。
    - ● 程序包括类定义和方法定义。
    - ● 大部分计算都通过对象的操作来表达。
    - ● 每个对象定义对应真实世界的某些对象或概念，而方法则对应真实世界中对象之间交互的方式。
