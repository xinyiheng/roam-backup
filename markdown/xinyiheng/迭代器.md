- 创建迭代器需要使用 iter() 这一内置函数来将可迭代对象进行转换，然后用 next() 方法来获取当中的迭代项。

>>> numbers = [1,2,3]
>>> iter_numbers = iter(numbers)
>>> next(iter_numbers)
1
>>> next(iter_numbers)
2
>>> next(iter_numbers)
3
>>> next(iter_numbers) #抛出StopIteration异常
