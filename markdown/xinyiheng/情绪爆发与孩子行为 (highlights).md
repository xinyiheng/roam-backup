- ### Highlights first synced by #Readwise [[December 2nd, 2025]]
    - 方法一：把“学习”和“错误”从“立刻改”改成“长期养”
    - 孩子的学习态度、自我管理能力、勇于认错，
      本质上都不是“教育问题”，而是“发育问题”。
    - 这意外地能让你轻松 30%：
      
      不是今天不认真 → 他将来完了
      
      而是他现在的执行功能还在发育 → 需要练习
      
      不是不承认错误 → 人格缺陷
      
      而是羞愧系统尚未成熟
      
      换一个角度，你会立刻温柔：
      
      他不是不愿意做得更好，他是还不会。
    - 方法二：用“稳定的原则 + 温和的态度”取代“情绪驱动的高强度纠正”
      比如：
      
      做事要专注，这是原则
      
      但当孩子做不到时，你用温和的语气（而不是情绪）
      
      这样孩子会更快学会责任，也更愿意说实话。
      
      原则坚定，方法柔软，这是最有效的组合。
    - 旧时代奖励压抑，新时代奖励心智成熟。
      我们那代是“压抑型强者”，
      孩子这代必须成为“心智型强者”。
