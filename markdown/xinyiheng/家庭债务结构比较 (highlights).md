- ### Highlights first synced by #Readwise [[December 12th, 2025]]
    - 股市本来应该有三个核心目的：
      
      为企业提供长期资本（融资功能）
      
      让社会财富长期增值（储蓄/投资功能）
      
      通过价格形成反映经济真实价值（信息功能）
      
      当一个国家的股市走偏，这三个功能会依次退化：
      
      价格剧烈波动 → 无法反映价值
      
      散户主导短线交易 → 无法稳定提供长期资本
      
      缺乏机构化长期持有者 → 市场成了“零和赌场”
      
      监管覆盖薄弱 → 炒作、操纵、概念虚火频繁出现
      
      企业融资成“一次性提款”，上市只是“圈钱”而不是长期经营
      
      最终市场的角色变成：
      
      不是企业成长的发动机，而是短期资金的竞技场。
      
      而这种 功能腐败 本身，正是长期资金不进入的原因。

    - 使用虚构人物的原因：
      不是为了制造戏剧
      
      不是为了写小说
      
      而是为了承载一种真实但无法被档案记录的时代体验
      
      这个人物承担的功能：
      作为读者进入真实历史的感知接口
      
      呈现普通人的理解方式、误解方式、犹豫方式
      
      连接宏观新闻与微观命运
    - 所有外部世界均为真实史实
      
      人物虚构，但时代真实。
      人物是镜头，不是主体。
    - 当你写作中产生自我怀疑时，可以回到这三句话
      我不是在替普通人辩护，我是在解释他的处境。
      
      我不是用虚构掩盖真实，而是用虚构照亮真实。
      
      我写的不是日本，而是任何一个普通人面对时代时的无力感。
      
      只要你不偏离这三句话，你就始终在正轨上。

    - 制度的空白，往往会被道德叙事填满。
    - 当你缺少：
      
      可执行的出清
      
      可追责的机制
      
      可惩罚的成本
      
      社会就只能靠：
      
      职业道德宣传
      
      价值观教育
      
      口号与自律
      
      但这种填补很脆弱，因为它对抗不了结构激励。
      
      所以这不是“道德是否重要”的问题，而是：
      
      道德应该是加分项，不应该是安全带。
