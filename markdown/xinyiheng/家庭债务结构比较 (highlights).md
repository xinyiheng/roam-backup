- ### Highlights first synced by #Readwise [[December 12th, 2025]]
    - 股市本来应该有三个核心目的：
      
      为企业提供长期资本（融资功能）
      
      让社会财富长期增值（储蓄/投资功能）
      
      通过价格形成反映经济真实价值（信息功能）
      
      当一个国家的股市走偏，这三个功能会依次退化：
      
      价格剧烈波动 → 无法反映价值
      
      散户主导短线交易 → 无法稳定提供长期资本
      
      缺乏机构化长期持有者 → 市场成了“零和赌场”
      
      监管覆盖薄弱 → 炒作、操纵、概念虚火频繁出现
      
      企业融资成“一次性提款”，上市只是“圈钱”而不是长期经营
      
      最终市场的角色变成：
      
      不是企业成长的发动机，而是短期资金的竞技场。
      
      而这种 功能腐败 本身，正是长期资金不进入的原因。
